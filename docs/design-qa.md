# Design QA-lista (manual)

Kort checklista för visuell regression inför release.

1. Mobil 360px: startvy (empty state) – inga layoutglapp eller överlapp.
2. Mobil 360px: scan-resultat – innehåll ryms, inga avklippta texter.
3. Mobil 360px: “For you” – kort/listor rullar och raderar inte layout.
4. Dark mode – alla vyer läsbara, ingen text försvinner.
5. Light mode – samma vyer som ovan, inga felaktiga kontraster.
6. Modal zoom – zoom in/ut fungerar och innehållet är fortfarande centrerat.
7. Buttons – disabled state ser tydligt inaktiv ut.
8. Buttons – active/hover/focus state ser tydligt aktiv ut.
9. Textkontrast – brödtext mot bakgrund uppfyller god läsbarhet.
10. Ikoner/badges – syns i både dark/light och har rätt padding.
